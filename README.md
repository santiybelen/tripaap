# Tripaap

Web app para organizar viajes en grupo (vuelos, hoteles, autos, excursiones, restaurantes) — todo en un solo lugar y compartido con quien viaja.

Estado: pre-scaffold. Próximo paso es arrancar con un Next.js mínimo deployable y agregar features de a una.

Stack planeado: Next.js + TypeScript + Tailwind, Postgres + Drizzle, Auth.js, deploy en Railway.
